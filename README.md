# OptiCrop
AI Based Smart Agriculture Optimization System 
## 📌 Project Summary

**OptiCrop** is an AI-powered Smart Agricultural Production Optimization Engine developed to help farmers make informed crop selection decisions using Machine Learning. The system analyzes important agricultural parameters such as Nitrogen (N), Phosphorus (P), Potassium (K), temperature, humidity, pH, and rainfall to recommend the most suitable crop for cultivation.

Instead of relying solely on traditional farming methods, **OptiCrop** provides intelligent, data-driven crop recommendations that improve productivity, optimize resource utilization, and support sustainable agricultural practices. The application features a simple and user-friendly web interface built with Flask, allowing farmers to easily access accurate crop predictions and make better farming decisions.

## ✨ Key Features

- AI-powered Crop Recommendation using Machine Learning
- Soil Nutrient Analysis (Nitrogen, Phosphorus, and Potassium)
- Weather-based Crop Prediction using Temperature, Humidity, pH, and Rainfall
- Fast and Accurate Crop Recommendation System
- User-Friendly Web Interface
- Flask-based Backend for Efficient Processing
- Responsive Web Design using HTML, CSS, and Bootstrap
- Data-driven Decision Support for Smart Farming

## 🏗️ Technical Architecture

The project follows a modular three-layer architecture:

### 1. Machine Learning Layer
Handles data preprocessing, feature engineering, and crop prediction using a trained Machine Learning model. The model analyzes soil nutrients and environmental conditions to recommend the most suitable crop for cultivation.

### 2. Backend Layer
Implemented using **Flask**, the backend manages user requests, processes input data, loads the trained Machine Learning model (`model.pkl`), and generates accurate crop recommendations.

### 3. Presentation Layer
Built with **HTML5, CSS3, Bootstrap, and Jinja2 Templates**, providing a responsive, user-friendly, and interactive web interface for farmers to enter agricultural data and view crop recommendations.

## 🚀 How to Run the Application

### 📌 Prerequisites

Before running the application, ensure the following software is installed on your system:

- Python 3.10 or higher installed and available in PATH
- pip (Python Package Manager)
- Flask
- Pandas
- NumPy
- Scikit-learn
- A modern web browser (Google Chrome, Microsoft Edge, or Mozilla Firefox)

---

### Step 1: Navigate to the Project Folder

The complete source code of the application is located inside the **OptiCrop** directory.

```bash
cd OptiCrop
```

---

### Step 2: Create a Virtual Environment (Optional but Recommended)

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Mac/Linux
source venv/bin/activate
```

---

### Step 3: Install Project Dependencies

Install all the required Python libraries.

```bash
pip install flask pandas numpy scikit-learn
```

---

### Step 4: Launch the Flask Web Application

Start the application using:

```bash
python app.py
```

---

### Step 5: Open the Application

Open your web browser and visit:

```text
http://127.0.0.1:5000
```

---

### 🧪 Testing (Optional)

Enter the required soil parameters such as Nitrogen (N), Phosphorus (P), Potassium (K), Temperature, Humidity, pH, and Rainfall. Click the **Predict** button to receive the most suitable crop recommendation generated by the Machine Learning model.

## 📂 Project Structure

- **1. Brainstorming & Ideation/** to **8. Project Demonstration/** : Contains project documentation and all project lifecycle phases.

- **OptiCrop/** : The main source code folder containing:

  - **app.py** : Flask web application that handles user requests and crop prediction.
  - **model.py** : Machine Learning model implementation and prediction logic.
  - **model.pkl** : Trained Machine Learning model used for crop recommendation.
  - **Crop_recommendation.csv** : Dataset used for training and testing the Machine Learning model.
  - **analysis.py** : Data preprocessing and analysis script.
  - **analysis.ipynb** : Jupyter Notebook for data analysis and model experimentation.
  - **templates/** : Contains HTML templates for the user interface.
  - **anaconda_projects/** : Project-related resources and supporting files.
